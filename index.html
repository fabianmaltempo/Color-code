Color code

Rules:

There is a pattern of five color pegs, duplicates are allowed. The codebreaker (you) tries to guess the pattern, in both order and color, within ten turns. Each guess is made by placing a row of code pegs on the decoding board. Once placed, press Guess and you will receive feedback from zero to five key pegs on the right of your attempt. A black key peg is placed for each code peg from the guess which is correct in both color and position. A gray key peg indicates the existence of a correct color code peg placed in the wrong position.
